# Sasol Reward Dashboard — Sandton

Welcome to the Sasol Reward Dashboard (Sandton) — a small, focused HTML-based front-end that tells the story of rewards at Sasol's Sandton location.

## Story
Once upon a time at Sasol Sandton, teams wanted to celebrate positive behaviour and outstanding performance. This dashboard was born as a simple, visual way to track rewards, recognise contributors, and share progress with colleagues. It presents key reward metrics, highlights recent winners, and provides an at-a-glance view so teams can stay motivated and connected.

## What it is
- A static HTML dashboard designed to be deployed to any static host (GitHub Pages, Netlify, S3, etc.).
- Designed for clarity and quick updates — add or update HTML files to change content.
- Language: HTML (simple, self-contained UI).

## How to view
1. Clone the repository:
   ```bash
   git clone https://github.com/SasolHub/sasol-reward-dashboard-sandton.git
   ```
2. Open `index.html` in your browser, or publish the repo with GitHub Pages to serve it.

## How to contribute
- Update the HTML files for content changes.
- If you want to add styles or interactivity, consider adding CSS and JavaScript files and including them in the HTML.
